# 📱 Shayari App (iOS)

A simple and elegant iOS application to explore, read, and share Gujarati Shayari. This app is designed for Shayari lovers who want a clean and smooth experience.

---

## 🚀 Features

- 📜 Browse Shayari by categories (Love, Sad, Friendship, Attitude, etc.)
- 🔍 Easy navigation and clean UI
- ❤️ Mark Shayari as Favorite
- 📤 Share Shayari via WhatsApp, Instagram, etc.
- 📋 Copy Shayari to clipboard
- 🌐 Offline support (no internet required)

---

## 🛠 Tech Stack

- Language: Swift
- Framework: UIKit / SwiftUI
- Architecture: MVVM (recommended)
- Storage: Local JSON / CoreData
- Minimum iOS Version: iOS 13+
